Filter out the geese

Question


	Write a function that takes a list of strings as an argument 
	and returns a filtered list containing the same elements but 
	with the 'geese' removed.

	The geese are any strings in the following array, which is 
	pre-populated in your solution:

	The elements in the returned array should be in the same order 
	as in the initial array passed to your function, albeit with 
	the 'geese' removed. Note that all of the strings will be in 
	the same case as those provided, and some elements may be repeated.

				

	["African", "Roman Tufted", "Toulouse", "Pilgrim", "Steinbacher"]
	For example, if this array were passed as an argument:

	 ["Mallard", "Hook Bill", "African", "Crested", "Pilgrim", "Toulouse", "Blue Swedish"]
	Your function would return the following array:

	["Mallard", "Hook Bill", "Crested", "Blue Swedish"]

Solution by -

My Solution


	function gooseFilter (birds) {
		var geese = ["African", "Roman Tufted", "Toulouse", "Pilgrim", "Steinbacher"];
		return birds.filter(x => !geese.includes(x))
		};
